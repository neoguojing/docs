# SLAM 技术手册（Visual / Stereo / RGBD / LiDAR）

版本：1.0

------------------------------------------------------------------------

# 1. SLAM 概述

SLAM（Simultaneous Localization and
Mapping）指机器人在未知环境中同时完成：

-   **定位（Localization）**
-   **地图构建（Mapping）**

输入：

-   相机
-   激光雷达
-   IMU

输出：

-   机器人位姿
-   环境地图

常见 SLAM 类型：

  类型                   传感器
  ---------------------- -------------
  Visual SLAM            单目 / 双目
  RGB‑D SLAM             深度相机
  LiDAR SLAM             激光雷达
  Visual‑Inertial SLAM   相机 + IMU

------------------------------------------------------------------------

# 2. SLAM 系统整体架构

现代 SLAM 一般采用 **Front‑End + Back‑End** 架构。

    Sensor Data
        │
        ▼
    Front-End
      - 特征提取
      - 帧间匹配
      - 位姿估计
        │
        ▼
    Loop Closure
        │
        ▼
    Back-End
      - 图优化
        │
        ▼
    Map Update

------------------------------------------------------------------------

# 3. 相机校正（Camera Calibration）

目的：

-   去除镜头畸变
-   建立双目几何关系

步骤：

## 去畸变（Undistortion）

去除：

-   径向畸变
-   切向畸变

常见工具：

-   OpenCV
-   相机驱动

## 极线校正（Rectification）

将左右图像的极线对齐，使匹配只需要进行**水平搜索**。

------------------------------------------------------------------------

# 4. 特征提取（Feature Extraction）

SLAM 常见特征：

  特征   类型     优点   缺点
  ------ -------- ------ ------------
  ORB    二进制   快     稳定性一般
  SIFT   浮点     稳定   计算慢
  SURF   浮点     较快   专利限制

## Keypoints

关键点是图像中稳定且可重复检测的位置，例如：

-   角点
-   纹理区域

常见检测算法：

-   FAST
-   DoG
-   Harris

## Descriptor

描述子用于描述关键点附近的图像纹理信息。

作用：

-   在不同图像中找到相同特征点。

常见描述子：

-   ORB Descriptor
-   SIFT Descriptor

------------------------------------------------------------------------

# 5. 立体匹配（Stereo Matching）

目标：计算左右图像之间的 **视差（Disparity）**。

公式：

d = x_left − x_right

常见方法：

  方法                   特点
  ---------------------- ----------
  Block Matching         快
  Semi Global Matching   精度高
  Feature Matching       稀疏匹配

------------------------------------------------------------------------

# 6. 三角化（Triangulation）

利用视差计算深度：

Z = f × B / d

参数说明：

-   Z：深度
-   f：焦距
-   B：双目基线
-   d：视差

特点：

-   视差越小，深度越远

------------------------------------------------------------------------

# 7. 帧间匹配（Frame Matching）

目标：估计相机运动。

流程：

1.  Descriptor Matching
2.  RANSAC 去除外点
3.  位姿估计

## Descriptor Matching

常见方法：

  方法          描述
  ------------- ------------
  Brute Force   暴力搜索
  FLANN         近似最近邻

ORB 常用：

-   Hamming Distance

## RANSAC

RANSAC（Random Sample Consensus）用于去除错误匹配。

流程：

1.  随机选择样本
2.  估计模型
3.  计算内点数量
4.  重复多次
5.  选择最优模型

------------------------------------------------------------------------

# 8. 回环检测（Loop Closure）

目标：

检测机器人是否回到之前的位置。

意义：

-   修正累计误差
-   保证地图一致性

## Bag of Words

思想：

将图像转换为**视觉单词**。

流程：

特征 → 聚类 → 视觉词典

相似度计算：

-   Cosine Similarity

## 几何验证

回环检测后需要进行几何验证：

-   Feature Matching
-   RANSAC

避免错误回环。

------------------------------------------------------------------------

# 9. 激光 SLAM

核心：**扫描匹配（Scan Matching）**

目标：估计

-   Δx
-   Δy
-   Δθ

------------------------------------------------------------------------

# 10. ICP

ICP（Iterative Closest Point）用于点云对齐。

优化目标：

min Σ \|\|p − Tq\|\|²

优点：

-   实现简单
-   精度高

缺点：

-   容易陷入局部最优

------------------------------------------------------------------------

# 11. CSM

Canonical Scan Matcher

特点：

-   Point‑to‑Line 匹配

优点：

-   更稳定
-   适合 2D 激光

------------------------------------------------------------------------

# 12. NDT

Normal Distribution Transform

思想：

将地图表示为高斯分布。

优点：

-   对噪声鲁棒
-   适合大规模地图

缺点：

-   计算量较大

------------------------------------------------------------------------

# 13. 扫描匹配算法对比

  算法   匹配方式   优点   缺点
  ------ ---------- ------ ----------
  ICP    点对点     简单   局部最优
  CSM    点对线     稳定   复杂
  NDT    概率模型   鲁棒   计算量大

------------------------------------------------------------------------

# 14. 图优化（Pose Graph Optimization）

SLAM 后端核心。

图结构：

-   Node：机器人位姿
-   Edge：位姿约束

优化目标：

min Σ eᵀ Ω e

其中：

-   e：误差
-   Ω：信息矩阵

------------------------------------------------------------------------

# 15. 优化算法

常见算法：

  方法                  特点
  --------------------- ------
  Gauss‑Newton          快
  Levenberg‑Marquardt   稳定

常见库：

-   g2o
-   Ceres Solver
-   GTSAM

------------------------------------------------------------------------

边包括：

-   里程计约束

# 17. Bundle Adjustment

同时优化：

-   相机位姿
-   三维地图点

目标：

最小化 **重投影误差**。
